# AI-Generated Text Detection using RoBERTa (PyTorch)

This project implements a transformer-based NLP classifier to detect AI-generated text using a fine-tuned RoBERTa model.

## Model Performance

Accuracy: 95.11%  
Precision: 92.15%  
Recall: 98.62%  
F1-score: 95.28%

## Technologies Used

Python  
PyTorch  
Hugging Face Transformers  
RoBERTa  

## Dataset

Combined multiple datasets containing human-written and AI-generated text samples and performed preprocessing before supervised fine-tuning.
